# Interactive Portfolio

A static, GitHub Pages-friendly portfolio website. It includes featured work filtering, project detail modals, an animated canvas hero, resume highlights, about section, theme toggle, and a contact form that opens a composed email.

## Publish on GitHub Pages

1. Push these files to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to deploy from your main branch and the repository root.
4. Replace the placeholder project, resume, and contact details in `index.html` and `script.js`.

No build step is required.
